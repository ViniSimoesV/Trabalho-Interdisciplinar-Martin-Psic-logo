# Martin Psicólogo 

Desenvolver uma aplicação web voltada para gestão de consultório psicólogo que realize o agendamento de consultas e o controle financeiro do consultório. A solução busca automatizar tarefas, que hoje, são realizadas de forma manual proporcionando maior praticidade para o psicólogo e seus pacientes
